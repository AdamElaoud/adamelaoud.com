Project Sync

Ongoing project


Element concept art

Project Sync was born from a personal challenge to design a game using core elements opposite to those of the most successful games in its genre i.e. Hearthstone, Duelsyt, etc. Project Sync is a TCG that:

  • is synchronusly turn-based
  • has no minion usage
  • has no basic resource cost (mana, etc)
  • is adaptable in-game as well as outside
  • allows one card played per turn (with some exceptions)
  • promotes fast-paced gameplay
